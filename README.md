# CodeAlpha_HeartDiseasePrediction
Heart Disease Prediction using Machine Learning (CodeAlpha Internship Task)


**Task 4: Disease Prediction from Medical Data**

- Used UCI Heart Disease Dataset (303 patients)
- Trained 4 models: Logistic Regression, Random Forest, SVM, XGBoost
- Best Accuracy: ~88-92% (Random Forest / XGBoost)
- Features: Age, Sex, Chest Pain, Cholesterol, etc.

**Project done as part of CodeAlpha Machine Learning Internship**
